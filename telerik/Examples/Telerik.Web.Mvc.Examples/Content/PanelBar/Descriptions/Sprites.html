This example shows how to configure the Telerik PanelBar for ASP.NET MVC to use sprites.

Use SpriteCssClasses method to set the css class name. The panelbar will render <span> tag with provided css class before item's text: 

<%= Html.Telerik().PanelBar()
        .Name("PanelBar")
        .Items(panelbar => 
        {
            panelbar.Add()
                    .Text("Telerik Extensions for ASP.NET MVC")
                    .SpriteCssClasses("icon-mvc")
                    .Items(parent =>
                    {
                        parent.Add().Text("Grid")
                              .SpriteCssClasses("icon-mvc-grid");
                        ...
                    }
        ...
        });
%>
